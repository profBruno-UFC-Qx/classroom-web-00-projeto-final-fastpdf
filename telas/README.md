# Telas implementadas

Esta pasta contém as telas referentes aos protótipos do sistema **FastPDF**.  
A seguir estão as telas escolhidas e uma breve descrição do que cada uma representa.

- **Tela de login**  
  ![Tela de login](tela%20login.png)  
  Tela responsável por permitir que o usuário informe suas credenciais e acesse o sistema.

- **Tela de recuperação de senha**  
  ![Tela de recuperação de senha](tela%20recuperacao%20senha.png)  
  Tela usada quando o usuário esquece a senha, permitindo solicitar a recuperação/alteração do acesso.

- **Tela de dashboard**  
  ![Tela de dashboard](tela%20dashboard.png)  
  Tela inicial após o login, exibindo um resumo e atalhos para as principais funcionalidades do sistema.

- **Tela de gestão de alunos**  
  ![Tela de gestão de alunos](tela%20gestao%20alunos.png)  
  Tela para visualizar e gerenciar alunos cadastrados, com opções de consulta e manutenção dos dados.

- **Tela de cadastro de aluno (manual)**  
  ![Tela de cadastro de aluno (manual)](tela%20cadastro%20aluno%20manual.png)  
  Tela destinada ao cadastro manual de um aluno, preenchendo os campos necessários no formulário.

- **Tela de importação de CSV (aluno)**  
  ![Tela de importação de CSV (aluno)](Tela%20importacao%20csv%20aluno.png)  
  Tela que permite importar uma lista de alunos via arquivo CSV para agilizar o cadastro em lote.

- **Tela de gestão financeira**  
  ![Tela de gestão financeira](Tela%20Financeiro.png)  
  Tela para acompanhar e administrar informações financeiras, como registros, valores e status.

- **Tela de configuração**  
  ![Tela de configuração](tela%20configuracao.png)  
  Tela onde o usuário pode ajustar configurações do sistema e preferências gerais.
